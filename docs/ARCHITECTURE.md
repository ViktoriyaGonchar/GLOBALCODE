# Архитектура GLOBALCODE

## Общая архитектура

GLOBALCODE построен на основе современного full-stack подхода с разделением на frontend и backend.

## Структура проекта

```
GLOBALCODE/
├── frontend/          # Next.js приложение (React)
├── backend/           # Express.js API сервер
├── shared/            # Общий код (типы, константы)
├── docs/              # Документация
└── docker/            # Docker конфигурации
```

## Технологический стек

### Frontend
- **Next.js 14+** - React фреймворк с App Router
- **TypeScript** - Типизация
- **Tailwind CSS** - Стилизация
- **Zustand** - Управление состоянием
- **React Query** - Работа с API
- **Monaco Editor** - Редактор кода
- **Video.js** - Видеоплеер

### Backend
- **Node.js 20+** - Runtime
- **Express.js** - Web фреймворк
- **TypeScript** - Типизация
- **Prisma** - ORM
- **PostgreSQL** - База данных
- **Redis** - Кэширование
- **Socket.io** - WebSocket для real-time

## Модули системы

### 1. Модуль проектов
- Хранение файлов в S3/MinIO
- Версионирование через Git-like систему
- Sandbox для запуска демо (WebContainers)

### 2. Модуль форума
- Иерархическая структура сообщений
- Система репутации
- Модерация контента

### 3. Модуль сообщений
- Real-time чат через WebSocket
- Групповые и приватные беседы

### 4. Модуль команд
- Совместная работа над проектами
- Управление задачами (Kanban)
- Календарь событий

### 5. Модуль образования
- Интерактивные курсы
- IDE-тренажёры
- Система сертификатов

### 6. Модуль видеотеки
- Видео хостинг
- Адаптивное качество
- Прогресс просмотра

## Безопасность

- JWT для аутентификации
- Rate limiting
- CORS настройки
- Helmet для безопасности заголовков
- Валидация входных данных (Zod)
- Хеширование паролей (bcrypt)

## Масштабирование

- Горизонтальное масштабирование через Docker
- Кэширование через Redis
- CDN для статических файлов
- Load balancing

## Мониторинг

- Логирование (Winston/Morgan)
- Error tracking (Sentry)
- Метрики (Prometheus)
- Health checks
